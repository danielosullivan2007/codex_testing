# Blackjack Web App

This repository contains a lightweight Blackjack game implemented with vanilla HTML, CSS, and JavaScript. Open `index.html` in a modern browser to play a solo round against the dealer.

## Features

- Shuffled 52-card deck with automatic reshuffle when depleted
- Dealer stands on soft 17, Aces count as 1 or 11
- Buttons for Deal, Hit, Stand, and Reset
- Animated card styling with responsive layout
- Accessible status updates and live region messaging

## Getting Started

1. Clone the repository.
2. Open `index.html` in your browser.
3. Click **Deal** to start a hand and use **Hit** or **Stand** to play.

Enjoy your game!
